### fuso 扩展仓库

fuso.cna 用于cobalt strike 的插件 
在cobaltstrike 创建bin 目录 将fuc.exe 放入 
然后在CS脚本管理器 中导入该脚本 即可使用fuso cobaltstrike插件实现一键转发
